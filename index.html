SELLL

SELL no forex é a ação de vender um determinado par de moedas esperando que seu valor diminua para obter lucro.

StopLoss

StopLoss no forex é um mecanismo usado para limitar as perdas em uma posição aberta.

BUYYY

Buy no forex significa comprar uma determinada moeda esperando que o seu valor aumente no futuro para obter lucro.

TakeProfit

TakeProfit é um tipo de ordem de saída em Forex que permite que os traders fechem uma posição quando alcançam um determinado nível de lucro desejado.
